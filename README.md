AI Wallet

An AI-powered non-custodial crypto wallet built with React.

Vision

AI Wallet lets users manage crypto using natural language. Instead of navigating complex wallets, users can simply tell the AI what they want to do.

Planned Features

- Non-custodial wallet
- AI Assistant
- Portfolio tracking
- Token swap
- Cross-chain bridge
- Scam detection
- Transaction history
